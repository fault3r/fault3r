# ps

**Process Status** > display information about running processes.
---

` ps [OPTION]... [PID] `
---

| **OPTION** | description |
|:---:|:---:|
|  | show processes running in the current shell |
| -e | all processes |
| -a, a | processes with ttys |
| -x | processes without ttys |
| -u | user-oriented format |
| -f | full format |
| -l | long format |
| -H, f | hierarchical tree format |

## Examples
` ps axu `
