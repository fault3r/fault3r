# gpasswd

**Group Password** > manage group memberships.
---

` gpasswd [OPTION] [GROUPNAME] `
---

| **OPTION** | description |
|:---:|:---:|
| -a, --add [USERNAME] | add a user to the specified group |
| -d, --delete [USERNAME] | remove a user from the specified group |

## Examples
` sudo gpasswd -a t-user testgroup `
