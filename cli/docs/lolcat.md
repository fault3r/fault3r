# lolcat

add a colorful rainbow effect to text output in the terminal.
---

` lolcat [OPTION]... `
---

| **OPTION** | description |
|:---:|:---:|
| -a | enable animation |
| -d | animation duration (default: 12) |
| -s | animation speed (default 20) |

## Examples
` ls -l | lolcat `

` tree | lolcat -a -s 200 `
