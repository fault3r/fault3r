# head

display the beginning of files.
---

` head [OPTION] [FILE]... `
---

| **OPTION** | description |
|:---:|:---:|
| -n , --lines [NUMBER] | print the first `NUMBER` lines instead of the first 10 |
| -c , --bytes [NUMBER] | print the first `NUMBER` bytes |

## Examples
` head -c 26 file.txt `
