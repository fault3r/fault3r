# cbonsai

random bonsai tree generator.
---

` cbonsai [OPTION]... `
---

| **OPTION** | description |
|:---:|:---:|
| -p | print tree to terminal when finished |
| -l | live mode |
| -i | infinite mode |
| -t [SECOND] | wait between steps of growth (default: 0.03) |
| -L [0-200] | more growth (default: 32) |

## Examples
` cbonsai -lp `
