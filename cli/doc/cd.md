# cd

**Change Diredctory** > change the shell working directory.
---

`cd [PATH]`
---

| **PATH** | description |
|:---:|:---:|
| | home directory |
| ~ | home directory |
| / | root directory |
| . | current directory |
| .. | parent directory |
| - | previous directory |

#### use ` \ ` instead of space, ` cd /home/linux\ terminal/ `

## Examples
` cd ../../home/ `

` cd /usr/share/ `
